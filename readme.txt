Bootloader for Arduino Primo and Primo core has been created using Nordic SDK11.
Starting from example bootloader in examples->dfu->bootloader directory,
replace the main file in this repository with that in the folder and compile
with settings in folder pca10040->dual_bank_ble_s132.
